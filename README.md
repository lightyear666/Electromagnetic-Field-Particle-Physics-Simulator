     ）。左侧菜单栏包含可选材料：矩形电场，矩形磁场，平面，粒子发射器。点击后在窗口左下方编辑属性：公有属性：拖动位置，
     旋转角度。电场（强度f(t)），磁场(f(t))，平面(默认有碰撞，宽度，高度)，粒子发射器（几种基本粒子，其他（电荷量），例
     如粒子1-发射时间，列表）。2D视角是垂直纸面向内，因此平面绘制成矩形，磁场电场采用标准画法（点阵和x阵，细线箭头）。窗
     口最下方是动画时间轴（t/s），玩家搭建场景后，可以点击播放/暂停和拖动条。根据以上基本需求，设计完善。并编码实现。
